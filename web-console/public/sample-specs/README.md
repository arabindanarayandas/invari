# Sample OpenAPI Specifications

This folder contains sample OpenAPI/Swagger specs for testing FluxGuard.

## Available Specs

### 1. stripe-api.json
A payment processing API with endpoints for:
- Creating charges
- Managing customers
- Processing refunds

### 2. user-management.yaml
A user management system with endpoints for:
- CRUD operations on users
- Password management
- User listing and filtering

### 3. ecommerce.json
An e-commerce platform API with endpoints for:
- Product management
- Order processing
- Shopping cart operations

## Usage

These specs can be uploaded when creating a new application in FluxGuard to test the platform's capabilities.
