# Audio Files for Invari Demo

This folder contains audio files that demonstrate the latency impact of voice AI agents:

- **fail.mp3**: Simulates a 3-second awkward silence when a request fails or requires retry
- **success.mp3**: Demonstrates the instant response when Invari auto-repairs the request

## Recording Instructions

To create these files, you can:

1. **fail.mp3**: Record or generate a 3-second silence followed by error tone
2. **success.mp3**: Record or generate a quick success chime (< 0.5 seconds)
